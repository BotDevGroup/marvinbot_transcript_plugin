# Marvinbot Transcript Plugin

Converter voice note to text with google api.

# Requirements

-   A working [Marvinbot](https://github.com/BotDevGroup/marvin) install

# Getting Started

## settings.json

```
{
    ...
    "plugin_configuration": {
        "marvinbot_transcript_plugin": {
            "key": "GOOGLE API KEY"
        }
    }
    ...
}

```
